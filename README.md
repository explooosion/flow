# flow

## Blog
[NPM - flow-bin 檢查JS](https://dotblogs.com.tw/explooosion/2016/08/03/235428)　
　
　

　　　
　　　　
## Clone
```bash
git clone https://github.com/explooosion/flow.git
```
　
　　
## Into Project
```bash
cd flow
```
　
## Install
```bash
npm install
```
　
## Run
```bash
npm run flow_check
```
